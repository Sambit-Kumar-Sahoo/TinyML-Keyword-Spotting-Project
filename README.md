# Real-Time AI Voice Recognition System

**Status:** Active Development  
**Tech Stack:** Python, TensorFlow/Keras, Librosa, NumPy, Pandas, Scikit-Learn

## Overview
This project is a TinyML pipeline designed to run keyword spotting on an ESP32 microcontroller. The goal is to build a lightweight Convolutional Neural Network (CNN) capable of processing raw 16kHz audio streams and identifying specific target commands ("On", "Off"), all within the limits of edge hardware.

Dataset Procurement: https://www.kaggle.com/datasets/neehakurelli/google-speech-commands
